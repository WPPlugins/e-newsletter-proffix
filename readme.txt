Plugin für die einfache Nutzung des PROFFIX E-Newsletter Services in Wordpress.


== Description ==

Mit diesem Plugin kann der E-Newsletter Service von PROFFIX einfach und übersichtlich in Wordpress eingebunden werden.

Neben einem individualisierbaren Widget ist es auch möglich, Anmeldeformulare mittels Shortcodes einzubinden.


*   Anmeldeformular für E-Newsletter als Widget
*   Anmeldeformular für E-Newsletter als Shortcode
*   Vollständig indivdualisierbar (E-Service, PROFFIX-Datenbank, Liste/n)


= Weitere Informationen =


Hier finden Sie weitere Infos zum [E-Newsletter Plugin für PROFFIX](https://www.pitw.ch/proffix/e-newsletter-proffix/) oder zum [PROFFIX E-Newsletter Service](http://www.proffix.net/Die-Software/Uebersicht-Modularitaet/Zusatzmodule/CRM)


== Installation ==

Entweder direkt über das Menu **Plugin** in Wordpress installieren oder aber die Installation manuell durchführen:

1. Das Archiv `proffix-newsletter.zip` nach `/wp-content/plugins/` entpacken.
2. Das Plugin über das **Plugin** Menu in WordPress aktivieren

Anschliessend kann die E-Newsletteanmeldung für PROFFIX CRM entweder über den Shortcode [proffixnewsletter]
oder direkt über das passende Widget eingebunden werden.

== Frequently Asked Questions ==

= Was ist der PROFFIX E-Newsletter Service? =

Mit [PROFFIX CRM](https://www.pitw.ch/loesungen/proffix-crm/) können Sie der bestenden Adressdatenbank einfach Newsletter und Infomails versendet werden.
Der E-Newsletter Service ermöglicht die selbständige Anmeldung von Interessierten für den Newsletter.

= Wieso braucht man ein Plugin für die Anmeldung? =

Dieses Plugin kümmert sich um die gesamte Formularerstellung, Verifikation und Übermittlung. Es muss kein manuelles Anmeldeformular erstellt werden.

= Welche Angaben werden für die Nutzung benötigt? =

Für die Nutzung des E-Newsletter Plugins für PROFFIX benötigten Sie folgende Angaben:
* E-Newsletter URL (evtl. Port)
* PROFFIX Datenbank in welche Anmeldungen übermittelt werden
* Newsletter Liste (z.B. STD)

= Können mehrere Newsletter - Listen verwendet werden? =

Mit dem E-Newsletter Plugin für PROFFIX können auch auf derselben Seite mehrere Formulare eingebunden werden.
